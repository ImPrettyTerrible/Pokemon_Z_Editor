
Este script ha sido diseñado para inyectar el Ultimate Editor en el juego fanmade Pokémon Z V2.14 creado con RPG Maker y Pokémon Essentials. (Asumo, en cualquier caso, es posible que de haber futuras actualizaciones, el script funcione correctamente)
Permite editar IVs, EVs, naturaleza, habilidades, género, amistad y ver información completa de tus Pokémon directamente desde el juego y realizar las modificaciones en tiempo real mientras juegas.

------------------------------------------------------------------------------------------------------------------

REQUISITOS

Sistema operativo Windows. (Script pensado para ser ejecutado desde Windows).

Tener instalado Ruby (recomendado: versión 2.7 o superior). (Es un script de Ruby al fin y al cabo, dejo aquí el link oficial https://rubyinstaller.org/ el más comodo para Windows, solo hay que darle al gran botón rojo de descarga). El lugar de instalación de ruby en tu dispositivo es irrelevante, mientras instales ruby en la ruta predefinida del instalador todo correcto.

Contar con el juego Pokémon Z V2.14/V2.13 con su archivo Game.exe. (Game.exe es el ejecutable para poder abrir el juego, entre otros muchos archivos relevantes que espero que nadie modifique de otra manera porque entonces dudo del funcionamiento del juego y de mi modificador).

------------------------------------------------------------------------------------------------------------------

PASOS DE INSTALACIÓN

(RECOMIENDO INSPECCIONAR EL ARCHIVO PRIMERO. Abrir con Bloc de Notas o similar y echarle un vistazo, en la cabecera, las priemras líneas, hay información e instrucciones en caso de que se prefiera verlas ahí)

Copia el archivo del script (inject_editor_ULTIMATE.rb) en la misma carpeta donde se encuentra Game.exe.

Abre un terminal o CMD en esa carpeta, para ello:

Haz clic derecho dentro de la carpeta → “Abrir ventana de PowerShell aquí” o “Abrir ventana de CMD aquí” / "Abrir en Terminal".

Ejecuta el script con Ruby usando el siguiente comando (ajustando la ruta de tu archivo Scripts.rxdata):

ruby inject_editor_ULTIMATE.rb "C:\Usuarios\<TuUsuario>\Documentos\Pokemon Z V2.14\Pokemon Z\Data\Scripts.rxdata" 

Pulse "ENTER". Tras esto deberia aparecer algún tipo de mensaje en la terminal.

Importante: Sustituye <TuUsuario> y la ruta según corresponda en tu dispositivo.
Asegúrate de usar comillas en la ruta y sobretodo de que el script se aplique sobre Data/Scripts.rxdata, pues el objetivo que mi script busca para modificar y esa su ubicación por defecto.

------------------------------------------------------------------------------------------------------------------

VERIFICACIÓN

El script crea automáticamente un backup del archivo Scripts.rxdata llamado Scripts.rxdata.backup, igualmente recominedo realizar una copia manual antes de ejecutar el comando en terminal por si acaso.

Para saber si la inyección fue exitosa:

Abre el juego normalmente.

Si el juego tarda unos segundos más en iniciar y aparece un cuadro de texto indicando que el servicio se está inicializando, el script fue cargado correctamente.

Presiona F9 dentro del juego para abrir el Ultimate Editor. (F9 es el botón por defecto, el más mañoso puede cambiarlo a voluntad, si es motivo de preocupación puedo indicar como cambiarlo).

------------------------------------------------------------------------------------------------------------------

PROBLEMAS COMUNES

Si los mensajes al abrir el juego no aparecen o el F9 no funciona / menú no abre entonces:

Cierra el juego.

Repite la ejecución del script desde el terminal. (Pueba a cerrar el terminal y abrir uno nuevo y entonces ahí repetir el comando).

Asegúrate de que la ruta de Scripts.rxdata sea correcta. (Pues precisamente este es archivo que se precisa modificar para que la injección funcione).

Si aparece un error en el terminal: revisa que Ruby esté correctamente instalado y que estés usando la ruta completa al archivo Scripts.rxdata, si todo esto es correcto, de nuevo, repite el comando en terminal nuevo.

| IMPORTANTE |

Tras abrir terminal e introducir el comando (visto previamente) recomiendo no cerrar dicha terminal y así en caso de que al abrir el juego este no muestre mensajes como: "Editor Ultimate cargando..."
Cierre el juego y repita el comando anterior en la misma terminal (tocando la flecha de dirección "ARRIBA"  del teclado y una vez vea el comando darle "ENTER", haga esto una única vez y no vuelva a realizar esta operación).

SOLO REPITA ESTO EN CASO DE QUE EL EDITOR NO PAREZCA FUNCIONAR
